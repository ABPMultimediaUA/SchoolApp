Este documento representa las llamadas a la api que lleva a cabo cada opción de la interfaz de la aplicación.

La interfaz no está representada gráficamente pero sí esquemáticamente.

Las columnas representan los roles, y las filas las llamadas.

La columna de Login representa las llamadas que hace el login para comprobar si el usuario y contraseña introducidos coinciden y existen en la base de datos.

Las opciones del menú principal de cada rol se dividen mediante líneas blancas.

Puede que el nombre de la opción que sale en este documento no coincida con su nombre en la interfaz, pero hace la(s) misma(s) llamadas que las del esquema aquí presentado.

Este esquema complementaría a un conjunto de diseños de interfaces que de momento no está disponible
